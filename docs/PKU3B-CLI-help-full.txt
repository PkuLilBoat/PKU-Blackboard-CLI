## pku3b -h

A tool for PKU students to check their courses.

Usage: pku3b [COMMAND]

Commands:
  assignment    获取课程作业信息/下载附件/提交作业 [aliases: a]
  coursetable   获取个人课表 [aliases: ct]
  course        获取课程列表和菜单入口 [aliases: c]
  announcement  获取课程公告 [aliases: ann]
  document      [aliases: doc]
  find          按标题进行确定性查找 [aliases: f]
  search        跨课程搜索结构化内容 [aliases: sfind]
  tree          [aliases: t]
  video         获取课程回放/下载课程回放 [aliases: v]
  syllabus      选课操作 [aliases: s]
  ttshitu       图形验证码识别 [aliases: tt]
  bark          Bark通知设置 [aliases: b]
  thesis-lib    学位论文检索 [aliases: th]
  init          (重新) 初始化用户名/密码
  config        显示或修改配置项
  cache         查看缓存大小/清除缓存
  help          Print this message or the help of the given subcommand(s)

Options:
  -h, --help     Print help (see more with '--help')
  -V, --version  Print version


## pku3b assignment -h

获取课程作业信息/下载附件/提交作业

Usage: pku3b assignment [OPTIONS] <COMMAND>

Commands:
  list      查看作业列表，按照截止日期排序 [aliases: ls]
  download  下载作业要求和附件到指定文件夹下 [aliases: down]
  submit    提交课程作业 [aliases: sb]
  help      Print this message or the help of the given subcommand(s)

Options:
  -f, --force                强制刷新
      --markdown             输出 Markdown [aliases: --md]
      --otp-code <OTP_CODE>  手机令牌码。当需要使用 OTP 登录，但未提供此参数时，将会从命令行交互式读取 OTP 码。 [default: ""]
  -h, --help                 Print help


## pku3b assignment list -h

查看作业列表，按照截止日期排序

Usage: pku3b assignment list [OPTIONS]

Options:
  -a, --all       显示所有作业，包括已完成的
      --all-term  显示所有学期的作业（包括已完成的）
  -h, --help      Print help


## pku3b assignment download -h

下载作业要求和附件到指定文件夹下

Usage: pku3b assignment download [OPTIONS] [ID]

Arguments:
  [ID]  (Optionl) 作业 ID (ID 形如 `f4f30444c7485d49`, 可通过 `pku3b assignment list` 查看)

Options:
  -d, --dir <DIR>  文件下载目录 (支持相对路径) [default: .]
      --all-term   在所有学期的作业范围中查找
  -h, --help       Print help (see more with '--help')


## pku3b assignment submit -h

提交课程作业

Usage: pku3b assignment submit [ID] [PATH]

Arguments:
  [ID]    作业 ID (形如 `f4f30444c7485d49`, 可通过 `pku3b assignment list` 查看)
  [PATH]  提交文件路径

Options:
  -h, --help  Print help (see more with '--help')

## pku3b find -h

按标题进行确定性查找

Usage: pku3b find [OPTIONS] <QUERY>

Arguments:
  <QUERY>  标题关键词或完整标题

Options:
  -f, --force                        
      --kind <KIND>                  仅搜索某一种内容类型，例如 assignment/document/announcement
      --course-index <COURSE_INDEX>  仅在指定课程索引内搜索
      --all-term                     显示所有学期
      --markdown                     输出 Markdown [aliases: --md]
      --otp-code <OTP_CODE>          手机令牌码。当需要使用 OTP 登录，但未提供此参数时，将会从命令行交互式读取 OTP 码。 [default: ""]
  -h, --help                         Print help
